Description of the functions of the m-file.
 
1. mitdb2_cut4_1.m: This script randomly splits extracted Nb and Arr type ECG samples into a test set and 4 training sets.
2. mitdb2_wt_fir_3R_RRextra.m: This script extracts normal and abnormal type 3R ECG samples from MIT-BIH signals after wavelet threshold filtering and low-pass filtering.
3. mitdb2mat.m: This script converts the format of ECG data from MIT-BIH to mat format.
